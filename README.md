## Problems you may encounter 

<Accordion title="New user ? " icon="fa-triangle-exclamation">
   Here are some tips to avoid common issues when using Code::Blocks. First, make sure to install the version that includes the MinGW compiler, otherwise you won’t be able to compile or run your programs. If you get a “compiler not found” error, check your settings and ensure the toolchain is properly configured. On first use, always verify that a default compiler is selected in the settings so that the build and run functions work correctly. It is also recommended to create proper projects instead of saving single files randomly, and to keep all source files inside the project folder to avoid build errors.
